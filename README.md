## System dependencies

-  Python 3.8
-  Flask 2.0.1
-  PicImageSearch 2.1.1
-  environs 9.3.3

# setup environment
- rename .env.example -> .env and set 
[SAUCE_API](https://saucenao.com/user.php?page=search-api)

# Install
- `git clonehttps://github.com/Oreki-san/anime_source`
- `cd anisource-web`
- `pip3 install -r requirements.txt`
- `python3 run.py`
- `Releases`
